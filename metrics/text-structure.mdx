---
title: 'Text Structure Metrics'
description: 'Metrics that validate the format and structure of generated text'
---

Text structure metrics validate the format of generated text without evaluating its content quality. These metrics return binary values (0 or 1) and work with all languages.

## Available Metrics

### is_json_object

Checks if generated outputs can be parsed as JSON objects.

**Scores:** Binary 0 or 1 values, where 1 indicates valid JSON object.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import is_json_object

outputs = [
    '{"name": "John", "age": 30}',
    '[1, 2, 3]',
    'Not JSON'
]

result = is_json_object(outputs)
print(result.metric_values)  # [1, 0, 0]
```

### is_json_array

Checks if generated outputs can be parsed as JSON arrays.

**Scores:** Binary 0 or 1 values, where 1 indicates valid JSON array.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import is_json_array

outputs = [
    '[1, 2, 3]',
    '{"key": "value"}',
    '[{"name": "John"}, {"name": "Jane"}]'
]

result = is_json_array(outputs)
print(result.metric_values)  # [1, 0, 1]
```

### is_float

Checks if generated outputs can be parsed as floating point numbers, optionally within a min/max range.

**Scores:** Binary 0 or 1 values, where 1 indicates valid float.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `min` (float | None): Optional minimum valid float
- `max` (float | None): Optional maximum valid float
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import is_float

outputs = ["3.14", "42", "not a number", "100"]

# Check if valid floats
result = is_float(outputs)
print(result.metric_values)  # [1, 1, 0, 1]

# Check if floats in range [0, 10]
result = is_float(outputs, min=0, max=10)
print(result.metric_values)  # [1, 0, 0, 0]
```

### is_int

Checks if generated outputs can be parsed as integers, optionally within a domain of valid integers.

**Scores:** Binary 0 or 1 values, where 1 indicates valid integer.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `domain` (Iterable[int] | Container[int] | None): Optional domain of valid integers (e.g., `range(1, 11)` or `{1, 3, 5}`)
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import is_int

outputs = ["42", "3.14", "100", "seven"]

# Check if valid integers
result = is_int(outputs)
print(result.metric_values)  # [1, 0, 1, 0]

# Check if integers in range 1-50
result = is_int(outputs, domain=range(1, 51))
print(result.metric_values)  # [1, 0, 0, 0]

# Check if in specific set
result = is_int(outputs, domain={1, 42, 99})
print(result.metric_values)  # [1, 0, 0, 0]
```

### contains_all_strings

Checks if generated outputs contain all strings in a given list.

**Scores:** Binary 0 or 1 values, where 1 indicates all strings are present.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `strings` (list[str]): A list of strings to match
- `case_sensitive` (bool): Whether to match case sensitively. Default: False
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import contains_all_strings

outputs = [
    "The quick brown fox jumps over the lazy dog",
    "The quick brown fox",
    "A fast brown fox leaps"
]

required = ["quick", "brown", "fox"]

result = contains_all_strings(outputs, required)
print(result.metric_values)  # [1, 1, 0]

# Case-sensitive matching
result = contains_all_strings(
    ["The QUICK brown fox"],
    ["quick"],
    case_sensitive=True
)
print(result.metric_values)  # [0]
```

### contains_any_strings

Checks if generated outputs contain any strings in a given list.

**Scores:** Binary 0 or 1 values, where 1 indicates at least one string is present.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `strings` (list[str]): A list of strings to match
- `case_sensitive` (bool): Whether to match case sensitively. Default: False
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import contains_any_strings

outputs = [
    "I love Python programming",
    "JavaScript is popular",
    "I enjoy cooking"
]

languages = ["Python", "Java", "JavaScript"]

result = contains_any_strings(outputs, languages)
print(result.metric_values)  # [1, 1, 0]
```

### contains_regex

Checks if generated outputs partially contain a given regular expression.

**Scores:** Binary 0 or 1 values, where 1 indicates regex match found.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `regex` (str): The regular expression to match
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import contains_regex

outputs = [
    "Email me at john@example.com",
    "My phone is 555-1234",
    "No contact info here"
]

# Check for email pattern
email_pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b'
result = contains_regex(outputs, email_pattern)
print(result.metric_values)  # [1, 0, 0]
```

### matches_regex

Checks if generated outputs fully match a given regular expression.

**Scores:** Binary 0 or 1 values, where 1 indicates complete regex match.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `regex` (str): The regular expression to match
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import matches_regex

outputs = [
    "ABC-123",
    "ABC123",
    "ABC-123-EXTRA"
]

# Must match exact pattern
pattern = r'[A-Z]{3}-\d{3}'
result = matches_regex(outputs, pattern)
print(result.metric_values)  # [1, 0, 0]
```

### validation_fn

Checks if generated outputs are valid according to an arbitrary function.

**Scores:** Binary 0 or 1 values, where 1 indicates validation passed.

**Languages:** All

**Parameters:**
- `generated_outputs` (list[str] | str): The model generated output(s) to evaluate
- `valid_fn` (Callable[[str], bool]): A function that takes a string and returns a bool determining validity. The function can also raise an exception on failure.
- `prompts` (list[str] | str | None): Optional prompts

**Returns:** `MetricValue[int]`

**Example:**
```python
from langcheck.metrics import validation_fn
import re

outputs = [
    "john@example.com",
    "invalid-email",
    "jane@test.org"
]

def is_valid_email(text: str) -> bool:
    pattern = r'^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}$'
    return bool(re.match(pattern, text))

result = validation_fn(outputs, is_valid_email)
print(result.metric_values)  # [1, 0, 1]
```

**Complex validation example:**
```python
import json

def validate_user_json(text: str) -> bool:
    """Check if text is valid JSON with required user fields."""
    try:
        data = json.loads(text)
        required_fields = ["name", "email", "age"]
        return all(field in data for field in required_fields)
    except:
        return False

outputs = [
    '{"name": "John", "email": "john@test.com", "age": 30}',
    '{"name": "Jane"}',
    'Not JSON'
]

result = validation_fn(outputs, validate_user_json)
print(result.metric_values)  # [1, 0, 0]
```

## Importing Metrics

Import text structure metrics directly from `langcheck.metrics`:

```python
from langcheck.metrics import (
    is_json_object,
    is_json_array,
    is_float,
    is_int,
    contains_all_strings,
    contains_any_strings,
    contains_regex,
    matches_regex,
    validation_fn
)
```

## Use Cases

### Structured Output Validation

Ensure LLM outputs follow the expected format:

```python
from langcheck.metrics import is_json_object, contains_all_strings

# Validate JSON structure
json_outputs = generate_responses(prompts)
json_valid = is_json_object(json_outputs)

# Check for required fields
required_fields = ['"name"', '"email"', '"age"']
fields_present = contains_all_strings(json_outputs, required_fields)

for i, (valid, complete) in enumerate(zip(json_valid.metric_values, fields_present.metric_values)):
    if not valid:
        print(f"Output {i}: Invalid JSON")
    elif not complete:
        print(f"Output {i}: Missing required fields")
```

### Numerical Output Validation

Validate that LLM generates numbers in the expected range:

```python
from langcheck.metrics import is_float

prompts = ["Rate this product from 1-5"]
outputs = ["4.5", "10", "excellent"]

result = is_float(outputs, min=1, max=5)
for i, valid in enumerate(result.metric_values):
    if not valid:
        print(f"Invalid rating: {outputs[i]}")
```

### Format Compliance

Check if outputs follow specific format requirements:

```python
from langcheck.metrics import matches_regex

# Validate phone numbers
phone_pattern = r'\d{3}-\d{3}-\d{4}'
result = matches_regex(outputs, phone_pattern)

# Validate dates (YYYY-MM-DD)
date_pattern = r'\d{4}-\d{2}-\d{2}'
result = matches_regex(outputs, date_pattern)
```

### Custom Validation Logic

Implement complex validation rules:

```python
from langcheck.metrics import validation_fn

def validate_summary(text: str) -> bool:
    """Summary must be 1-3 sentences and under 200 characters."""
    sentences = text.count('.') + text.count('!') + text.count('?')
    return 1 <= sentences <= 3 and len(text) <= 200

result = validation_fn(summaries, validate_summary)
```

## Combining with Assertions

Text structure metrics work well with assertions for testing:

```python
from langcheck.metrics import is_json_object

outputs = generate_json_outputs(prompts)
result = is_json_object(outputs)

assert all(result.metric_values), "All outputs must be valid JSON objects"
```
