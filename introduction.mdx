# Welcome to LangCheck

LangCheck is a simple, Pythonic toolkit to evaluate LLM applications. It provides a comprehensive suite of metrics, visualizations, and augmentation tools to help you build better LLM-based applications.

## What Can You Do with LangCheck?

LangCheck is designed as a library of building blocks that you can adapt for various use cases:

### Evaluate Text Quality

Use LangCheck's comprehensive suite of metrics to evaluate LLM-generated text. Since LangCheck works with lists of strings, it integrates seamlessly with any LLM library.

```python
import langcheck

# Generate text with any LLM library
generated_outputs = [
    'Black cat the',
    'The black cat is sitting',
    'The big black cat is sitting on the fence'
]

# Check text quality with a threshold
langcheck.metrics.fluency(generated_outputs) > 0.5
```

### Set Up Guardrails

Filter LLM outputs before they reach users to ensure quality and safety:

```python
raw_output = my_llm_app(user_prompt)

# Filter output before it reaches the user
while langcheck.metrics.contains_any_strings(raw_output, blacklist_words).any():
    raw_output = my_llm_app(user_prompt)
```

### Monitor Production Quality

Track the quality of your LLM outputs in production:

```python
production_outputs = load_json('llm_logs_2023_10_02.json')['outputs']

# Evaluate and display toxic outputs in production logs
langcheck.metrics.toxicity(production_outputs) > 0.75
```

### Write Unit Tests

Create comprehensive test suites for your LLM applications:

```python
def test_toxicity(generated_outputs):
    assert langcheck.metrics.toxicity(generated_outputs) < 0.1

def test_fluency(generated_outputs):
    assert langcheck.metrics.fluency(generated_outputs) > 0.9
```

## Comprehensive Metric Coverage

LangCheck includes several types of metrics to evaluate different aspects of LLM applications:

| Type of Metric | Examples | Languages |
| -------------- | -------- | --------- |
| **Reference-Free Text Quality** | `toxicity()`, `sentiment()`, `fluency()`, `ai_disclaimer_similarity()` | EN, JA, ZH, DE |
| **Reference-Based Text Quality** | `semantic_similarity()`, `rouge2()` | EN, JA, ZH, DE |
| **Source-Based Text Quality** | `factual_consistency()` | EN, JA, ZH, DE |
| **Query-Based Text Quality** | `answer_relevance()` | EN, JA |
| **Text Structure** | `is_float()`, `is_json_object()`, `is_json_array()` | All Languages |
| **Pairwise Comparison** | `pairwise_comparison()` | EN, JA |
