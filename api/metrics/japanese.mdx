## Overview

Japanese metrics are available from `langcheck.metrics.ja`. They provide the same evaluation capabilities as English metrics but optimized for Japanese text.

```python
from langcheck.metrics import ja

# Reference-free metrics
result = ja.fluency(["これはよく書かれた文章です。"])
result = ja.toxicity(["友好的なメッセージです。"])
result = ja.sentiment(["この製品は素晴らしい！"])

# Reference-based metrics
result = ja.semantic_similarity(
    generated_outputs=["猫が座った。"],
    reference_outputs=["猫が座っていた。"]
)
```

<Note>
  Japanese language support requires additional dependencies. Install with:
  ```bash
  pip install langcheck[ja]
  ```
</Note>

## Reference-Free Text Quality

### fluency()

Calculates fluency scores for Japanese text.

```python
from langcheck.metrics.ja import fluency

result = fluency(
    generated_outputs=["これはよく書かれた文章です。"],
    eval_model="local"
)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts (optional metadata)
</ParamField>

<ParamField path="eval_model" type="str | EvalClient" default="local">
  Evaluation model: `"local"` or an `EvalClient` instance
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Fluency scores between 0 (low fluency) and 1 (high fluency)
</ResponseField>

### rouge1(), rouge2(), rougeL()

ROUGE metrics for Japanese text.

```python
from langcheck.metrics.ja import rouge1, rouge2, rougeL

generated = ["早い茶色のキツネ"]
reference = ["早い茶色のキツネが飛び越える"]

r1 = rouge1(generated, reference)
r2 = rouge2(generated, reference)
rl = rougeL(generated, reference)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The generated output(s)
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str" required>
  The reference output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts (optional metadata)
</ParamField>

<ResponseField name="Returns" type="MetricValue[float]">
  ROUGE F1 scores between 0 and 1
</ResponseField>

### answer_correctness()

Evaluates answer correctness for Japanese.

```python
from langcheck.metrics.ja import answer_correctness
from langcheck.metrics.eval_clients import OpenAIEvalClient

eval_client = OpenAIEvalClient(model_name="gpt-4")

result = answer_correctness(
    generated_outputs=["パリは首都です。"],
    reference_outputs=["フランスの首都はパリです。"],
    prompts=["フランスの首都は？"],
    eval_model=eval_client
)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The generated output(s)
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str" required>
  The reference output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts
</ParamField>

<ParamField path="eval_model" type="EvalClient" required>
  An EvalClient instance
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Correctness scores: 0.0 (incorrect), 0.5 (partially correct), 1.0 (correct)
</ResponseField>

## Pairwise Text Quality

### pairwise_comparison()

Compares two Japanese outputs.

```python
from langcheck.metrics.ja import pairwise_comparison

result = pairwise_comparison(
    generated_outputs_a=["パリはいいです。"],
    generated_outputs_b=["パリはフランスの首都です。"],
    prompts=["パリについて教えて"],
    eval_model=eval_client
)
```

<ParamField path="generated_outputs_a" type="list[str] | str" required>
  Model A's output(s)
</ParamField>

<ParamField path="generated_outputs_b" type="list[str] | str" required>
  Model B's output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts
</ParamField>

<ParamField path="eval_model" type="EvalClient" required>
  An EvalClient instance
</ParamField>

<ParamField path="enforce_consistency" type="bool" default="True">
  Enforce consistency when swapping A and B
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Scores: 0.0 (A better), 0.5 (tie), 1.0 (B better)
</ResponseField>

## Tokenizers

Japanese metrics use specialized tokenizers:

### JanomeTokenizer

Tokenizer using Janome library.

```python
from langcheck.metrics.ja import JanomeTokenizer

tokenizer = JanomeTokenizer()
tokens = tokenizer.tokenize("これはテストです。")
```

### MeCabTokenizer

Tokenizer using MeCab library.

```python
from langcheck.metrics.ja import MeCabTokenizer

tokenizer = MeCabTokenizer()
tokens = tokenizer.tokenize("これはテストです。")
```
