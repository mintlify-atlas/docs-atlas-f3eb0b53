## Overview

English metrics are available from `langcheck.metrics.en` or directly from `langcheck.metrics` for commonly used metrics.

```python
from langcheck.metrics import en

# Reference-free metrics
result = en.fluency(["This is a well-written sentence."])
result = en.toxicity(["This is a friendly message."])
result = en.sentiment(["I love this product!"])

# Reference-based metrics
result = en.semantic_similarity(
    generated_outputs=["The cat sat."],
    reference_outputs=["A cat was sitting."]
)
```

## Reference-Free Text Quality

Evaluate generated outputs without requiring reference text.

### fluency()

Calculates fluency scores measuring how well-written and grammatical the text is.

```python
from langcheck.metrics.en import fluency

result = fluency(
    generated_outputs=["This is well written.", "This txt bad"],
    eval_model="local"  # or OpenAIEvalClient instance
)
print(result.metric_values)  # [0.95, 0.42]
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts used to generate outputs (optional metadata)
</ParamField>

<ParamField path="eval_model" type="str | EvalClient" default="local">
  Evaluation model: `"local"` for local model or an `EvalClient` instance
</ParamField>

<ParamField path="local_overflow_strategy" type="str" default="truncate">
  Strategy for inputs exceeding local model length: `"nullify"`, `"truncate"`, or `"raise"`
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Fluency scores between 0 (low fluency) and 1 (high fluency)
</ResponseField>

## Reference-Based Text Quality

Evaluate outputs by comparing against reference text.

### semantic_similarity()

Calculates semantic similarity between generated and reference outputs.

```python
from langcheck.metrics.en import semantic_similarity

result = semantic_similarity(
    generated_outputs=["The cat sat on the mat."],
    reference_outputs=["A cat was sitting on a mat."],
    eval_model="local"
)
print(result.metric_values)  # [0.92]
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str" required>
  The reference output(s) to compare against
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts (optional metadata)
</ParamField>

<ParamField path="eval_model" type="str | EvalClient" default="local">
  Evaluation model: `"local"` or an `EvalClient` instance
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Similarity scores between 0 (different) and 1 (semantically identical)
</ResponseField>

### rouge1(), rouge2(), rougeL()

Calculates ROUGE overlap scores comparing generated vs. reference text.

```python
from langcheck.metrics.en import rouge1, rouge2, rougeL

generated = ["The quick brown fox jumps"]
reference = ["The quick brown fox jumps over the lazy dog"]

r1 = rouge1(generated, reference)
r2 = rouge2(generated, reference)
rl = rougeL(generated, reference)

print(r1.metric_values)  # [0.625] (unigram overlap)
print(r2.metric_values)  # [0.500] (bigram overlap)
print(rl.metric_values)  # [0.625] (longest common subsequence)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str" required>
  The reference output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts (optional metadata)
</ParamField>

<ResponseField name="Returns" type="MetricValue[float]">
  ROUGE F1 scores between 0 (no overlap) and 1 (perfect overlap)
</ResponseField>

### answer_correctness()

Evaluates if the answer is correct compared to a reference answer.

```python
from langcheck.metrics.en import answer_correctness
from langcheck.metrics.eval_clients import OpenAIEvalClient

eval_client = OpenAIEvalClient(model_name="gpt-4")

result = answer_correctness(
    generated_outputs=["Paris is the capital."],
    reference_outputs=["The capital of France is Paris."],
    prompts=["What is the capital of France?"],
    eval_model=eval_client
)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str" required>
  The reference (correct) output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts/questions that generated the outputs
</ParamField>

<ParamField path="eval_model" type="EvalClient" required>
  An EvalClient instance (e.g., OpenAIEvalClient)
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Correctness scores: 0.0 (incorrect), 0.5 (partially correct), 1.0 (correct)
</ResponseField>

## Source-Based Text Quality

Evaluate outputs in context of source documents.

### factual_consistency()

Checks if the generated output is factually consistent with the source documents.

```python
from langcheck.metrics.en import factual_consistency
from langcheck.metrics.eval_clients import OpenAIEvalClient

eval_client = OpenAIEvalClient()

result = factual_consistency(
    generated_outputs=["The Eiffel Tower is in Paris."],
    sources=["Paris is home to the famous Eiffel Tower."],
    prompts=["Where is the Eiffel Tower?"],
    eval_model=eval_client
)
```

<ParamField path="generated_outputs" type="list[str] | str" required>
  The model generated output(s) to evaluate
</ParamField>

<ParamField path="sources" type="list[str] | str" required>
  The source document(s) to check consistency against
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts (optional metadata)
</ParamField>

<ParamField path="eval_model" type="EvalClient" required>
  An EvalClient instance for evaluation
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Consistency scores: 0.0 (inconsistent), 0.5 (partially consistent), 1.0 (fully consistent)
</ResponseField>

## Pairwise Text Quality

Compare two model outputs to determine which is better.

### pairwise_comparison()

Compares two model outputs and returns which is preferred.

```python
from langcheck.metrics.en import pairwise_comparison
from langcheck.metrics.eval_clients import OpenAIEvalClient

eval_client = OpenAIEvalClient()

result = pairwise_comparison(
    generated_outputs_a=["Paris is nice."],
    generated_outputs_b=["Paris is the capital of France."],
    prompts=["Tell me about Paris"],
    eval_model=eval_client
)
print(result.metric_values)  # [1.0] means B is better, [0.0] means A is better, [0.5] is tie
```

<ParamField path="generated_outputs_a" type="list[str] | str" required>
  Model A's generated output(s)
</ParamField>

<ParamField path="generated_outputs_b" type="list[str] | str" required>
  Model B's generated output(s)
</ParamField>

<ParamField path="prompts" type="list[str] | str | None">
  The prompts used to generate outputs
</ParamField>

<ParamField path="sources_a" type="list[str] | str | None">
  Source documents for model A
</ParamField>

<ParamField path="sources_b" type="list[str] | str | None">
  Source documents for model B
</ParamField>

<ParamField path="reference_outputs" type="list[str] | str | None">
  Optional reference outputs
</ParamField>

<ParamField path="eval_model" type="EvalClient" required>
  An EvalClient instance
</ParamField>

<ParamField path="enforce_consistency" type="bool" default="True">
  If True, only return scores where swapping A and B gives consistent results
</ParamField>

<ResponseField name="Returns" type="MetricValue[float | None]">
  Comparison scores: 0.0 (A is better), 0.5 (tie), 1.0 (B is better)
</ResponseField>
